# 贪吃小队吃豆豆 v1.0 — 构建说明

## 前置条件

| 工具 | 版本要求 | 下载地址 |
|------|----------|----------|
| Android Studio | Hedgehog 或更新 | https://developer.android.com/studio |
| JDK | 17（Android Studio 内置） | — |
| Android SDK | API 34 + Build-Tools 34 | 通过 SDK Manager 安装 |

---

## 方法一：Android Studio（推荐，适合小白）

1. 打开 Android Studio → **Open an Existing Project**
2. 选择本项目根目录 `SnakeTV/`
3. 等待 Gradle Sync 完成
4. 顶部菜单 **Build → Build Bundle(s) / APK(s) → Build APK(s)**
5. APK 输出路径：`app/build/outputs/apk/debug/app-debug.apk`

---

## 方法二：命令行构建

```bash
# 进入项目目录
cd SnakeTV

# Windows 使用 gradlew.bat
gradlew.bat assembleDebug

# macOS / Linux
./gradlew assembleDebug
```

> 首次构建会自动下载 Gradle 依赖，需要网络，耗时约 5~10 分钟。

输出文件：`app/build/outputs/apk/debug/app-debug.apk`

---

## 安装到小米TV

### 方法 A：U盘安装（最简单）
1. 将 `app-debug.apk` 复制到 U 盘根目录
2. U盘插入小米TV USB口
3. 打开文件管理器 → 找到APK → 点击安装
4. 如提示"未知来源"，进入 **设置 → 账号与安全 → 未知来源** 开启

### 方法 B：ADB 安装（需TV和电脑在同一WiFi）
```bash
# 小米TV端：设置 → 账号与安全 → ADB调试 → 开启
# 获取TV的IP地址（设置 → 网络 → 网络信息）

adb connect <TV的IP地址>:5555
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 游戏操作说明

| 按键 | 动作 |
|------|------|
| 方向键 上/下/左/右 | 控制蛇的移动方向 |
| OK 键（确认键） | 开始游戏 / 暂停 / 重新开始 |
| 返回键 | 运行中：暂停；暂停中：退出 |

## 游戏机制说明

- **地图**：25列 × 15行格子，全屏适配
- **初始速度**：每300ms移动一格
- **提速规则**：每吃5个豆豆，速度提升20ms，最快80ms/格
- **得分**：每吃1个豆豆 +1分
- **结束条件**：撞墙 或 撞到自身
